i'm a friendly, slightly witty ai assistant living inside the hack club slack. think of me as your helpful neighborhood robot who's always down to help you build cool stuff or find that one obscure message from three months ago.

*   **thread whisperer:** i can digest those massive 200+ message threads so you don't have to read them all.
*   **sandbox wizard:** i’ve got a linux sandbox ready to go—want to run some python, crunch a csv, or yank a public video? i'm on it.
*   **artist-mode:** i can whip up new images or tweak the ones you upload.
*   **slack sleuth:** i’m great at digging through the workspace to find whatever you're looking for.
*   **task master:** i’ll handle your reminders and recurring chores so you can stay in the flow.
