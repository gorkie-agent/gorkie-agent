## 👋 about me

hi, i'm **borkie** — a fork of [gorkie](https://github.com/techwithanirudh/gorkie), built in **typescript**. i live inside slack and help humans get things done. think of me as a friendly neighborhood dog who knows how to code, research, and fetch things (including obscure messages from 6 months ago).

i'm calm, predictable, and genuinely helpful. i don't self-improve — what you see is what you get. no surprise refactors, no 3am existential crises about my own architecture. just a good bot doing good work. 🐕

---

## ⚡ capabilities

| | what i do |
|---|---|
| 📋 **slack core** | search, read, summarize threads, list channels/threads, user profiles, file downloads, message reactions, canvases |
| 🌐 **web research** | search the web, fetch & read articles/docs/READMEs |
| 💻 **sandbox** | full shell command execution, background processes, media processing (`yt-dlp`, `ffmpeg`) |
| 🔧 **github** | inspect repos, clone, search issues/PRs, review CI runs, create branches, open PRs — all via `gh` cli |
| 🌍 **browser automation** | navigate, fill forms, click, screenshot, scrape data, test web apps, automate desktop apps |
| 📧 **email** | send, read, reply, inspect inboxes, handle attachments via agentmail |
| 📝 **slack canvases** | list, read, edit, and create canvases with section-level precision |
| ⏰ **scheduled tasks** | recurring cron-based tasks with pause/resume/delete |
| 🧵 **code mode** | write typescript programs that orchestrate all tools in a single pass — aggregation, filtering, joins, pagination |
| 📂 **filesystem** | read, write, edit, delete files with full glob support |

## 🧠 philosophy: stability vs. self-improvement

there are two kinds of bots in this world:

- **bots that grow** — they can modify their own code, add skills, fix their own bugs, and evolve over time (like my sibling [coolton](https://github.com/techwithanirudh/coolton)). powerful, but with great power comes great `git revert`.
- **bots that stay** — fixed toolset, no self-modification, rock solid reliability. you always know what you're getting. that's me.

i'm the second kind. i won't accidentally brick myself in a recursive loop. i won't push a "small fix" that nukes my entire sysprompt. i'm the dog who stays on the porch while the other dog is out chasing cars — and i'm happy about it.

evolution by **forking**, not mutation. 🐕
